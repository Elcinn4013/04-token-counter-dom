  • How It Works
  1. 1

    User Input

    The user types a prompt into the interface.

  2. 2

    Tokenisation

    The input text is broken down into smaller pieces called tokens.

  3. 3

    Embedding

    Tokens are converted into numerical vectors that represent their meaning.

  4. 4

    Attention Layers

    The model analyzes the relationships and context between different tokens.

  5. 5

    Prediction

    The model calculates the probabilities for the next most likely token.

  6. 6

    Output Generation

    The predicted token is added to the sequence, and the process repeats.
